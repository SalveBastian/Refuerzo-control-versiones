# 🌐 Control de Versiones con Git y GitHub

Proyecto desarrollado como parte del programa **Análisis y Desarrollo de Software (ADSO)** del SENA, con el objetivo de aplicar un flujo profesional de trabajo usando Git y GitHub.

---

## 📋 Descripción

Aplicación web desarrollada con HTML, CSS y JavaScript que demuestra el uso completo de herramientas de control de versiones: commits atómicos, ramas, Pull Requests, Issues, tablero Kanban e integración continua.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Git & GitHub
- GitHub Actions (CI)

---

## 🚀 ¿Cómo ver el proyecto?

1. Clona el repositorio:
```bash
   git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
```
2. Abre el archivo `index.html` en tu navegador.

## 🔄 Flujo de trabajo

Este proyecto sigue el flujo profesional de Git:

- ✅ Commits atómicos con **Conventional Commits**
- ✅ Ramas por funcionalidad
- ✅ Pull Requests documentados
- ✅ Issues con criterios **Given/When/Then**
- ✅ Tablero **Kanban** en GitHub Projects
- ✅ Pipeline de **CI** con GitHub Actions
